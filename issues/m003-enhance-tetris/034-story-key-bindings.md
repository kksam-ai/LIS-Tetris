# Title
自定义快捷键系统

# Story
作为一个游戏玩家
我希望能够自定义游戏的控制按键
以获得更符合个人习惯的操作体验

# Acceptance Criteria
- [ ] 可以自定义所有游戏控制功能的按键
- [ ] 每个功能可以设置两个不同的按键（主按键和备用按键）
- [ ] 未设置时使用默认按键（例如：左移默认为←和H键）
- [ ] 设置界面清晰易用，主备按键设置分开显示
- [ ] 不允许按键冲突（同一个按键不能用于不同的功能）
- [ ] 可以清除任意功能的任一按键，但必须确保每个功能至少保留一个可用的按键
- [ ] 清除按键时有清晰的提示，防止误操作

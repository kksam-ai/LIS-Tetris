# Title
实现快捷键设置功能

# Introduction
实现游戏控制功能的快捷键自定义设置，允许用户为每个控制功能设置两个不同的按键（主按键和备用按键），可以清除任意一个按键，但必须确保每个功能至少保留一个可用的按键。

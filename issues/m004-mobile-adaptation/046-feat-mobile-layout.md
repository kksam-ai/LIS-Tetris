# Title
移动端游戏界面布局

# Introduction
重新设计游戏界面在移动端（竖屏）的布局，采用固定高度的三段式布局：
- 顶部区域固定高度150px，包含用户信息和游戏状态
- 中部区域自适应高度，显示主游戏画布
- 底部区域固定高度180px，用于触控按钮
通过合理的空间分配和布局优化，提供最佳的移动端游戏体验。

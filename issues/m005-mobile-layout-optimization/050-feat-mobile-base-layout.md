# Title
实现移动端基础上下布局

# Introduction
将移动端游戏界面调整为上下结构布局,上部为游戏主区域,下部为控制区域。
移除原有的固定高度设计,改为基于屏幕比例的自适应布局。控制区域占用游戏区域
确定后的所有剩余可视空间,按钮在控制区内顶部对齐,底部自然留白。

# Title

# Introduction
重新设计底部控制区的按钮布局,第一行从左到右依次为:左移、右移、硬降落、旋转；
第二行从左到右依次为:快速降落、重来、暂停、退出。通过计算合适的左边距,
使快速降落按钮在视觉上对齐第一行左移和右移按钮的中间位置。
